Apprendre les lignes de commandes sur Linux :

Les principales lignes de commandes à savoir :

Avant tout :
  • Ctrl + Alt + t =permet d'ouvrir le terminal
  • Ctrl + a = pour aller au début de la ligne
  • Ctrl + e = pour aller à la fin de la ligne
  • Ctrl + c = annule la dernière commande
  • sudo su : permet de passer en mode administrateur
    NB : en tapant cette commande il demande d'entrer votre mot de passe
  • passwd : cette commande permet d'entrer un mot de passe
  • clear : efface tout ce qu'il y a sur le terminal
  • (logicel) --version ou (logiciel) --v : permet de savoir la version du logiciel installé
  • cd (Change Directory) : cette commande permet de se placer sur un répertoire
    cd / : racine du répertoire
    cd .. : remonte au répertoire de là où vs êtes
  • ls ( List Directory) : liste les fichiers dans le répertoire dit
    ls -a : liste les fichiers cachés
    ls -l : liste les fichiers de manière détaillé
    ls -la : liste les fichiers cachés et les détaille
    NB : Pour aller plus vite, tapez ll.
  • touch : permet de créer un fichier vide (préciser l'extension du fichier exple : fichier.txt)
  • rm ( remove ) : permet de supprimer un fichier
    Exemple : rm (source) (destination)
  • cp (copy) : permet de copier un fichier
    Exemple : cp (source) (destination)
    • NB : pour la source, préciser le chemin (là où se trouve votre fichier)
  • mkdir ( Make Directory) : créer un répertoire
  • rmdir ( remove Directory) : supprime le répertoire
  • mv : correspond au « couper » sur windows = déplacer , renommer
  • cat : permet d'afficher le contenu d'un fichier.
  • nano : permet de modifier le fichier
  • - - help : permet de montrer l'aide d'une commande spécifique
  • ps : montre les processus en cours d'utilisation
  • pwd : cette commande permet d'afficher les répertoires de travail/ en cours d'utilisation
  • apt-get install "nom du paquet à installer" : permet d'installer des paquets, des logiciels compltR.
    apt-get update : permet de télécharger toutes les mises à jours
    apt-get upgrade : permet de les installer
    apt-get purge "nom du logiciel" : efface le logiciel installé
  • chmod : permet de donner les droits ( de lecture, d'écriture et/ou d’exécution ) à un fichier.
    Exemple : chmod 777 "nom du fichier" : ne donne aucun droits (de lecture, d'écriture et/ou d’exécution) sur le fichier « sauf pour l'admin »
  • exit : permet de sortir du terminal
  • shutdown "nb de minutes": éteint l'ordinateur dans x minutes.
  • apropos + (commande) : indique le rôle d'une commande
  • tar.gz : décompresse les fichiers d'installations
